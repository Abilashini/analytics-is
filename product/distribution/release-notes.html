WSO2 IS development team is pleased to announce the release of WSO2 IS Analytics 5.2.0

WSO2 IS Analytics 5.2.0 combines real-time, batch, interactive, and predictive (via machine learning) analysis of data into one integrated platform to support the multiple demands of comprehensive WSO2 IS use cases.

WSO2 IS Analytics is based on WSO2 Data Analytics Server 3.1.0, and is powered by WSO2 Carbon, the SOA middleware component platform.

An open source product, WSO2 Carbon is available under the Apache Software License (v2.0)

You can download this distribution from http://wso2.com/products/identity-server and give it a try.

How to Run

  1. Extract the downloaded zip
  2. Go to the bin directory in the extracted folder
  3. Run the wso2server.sh or wso2server.bat as appropriate
  4. Point your browser to the URL https://localhost:9444/carbon
  5. Use "admin", "admin" as the username and password to login as an admin
  6. If you need to start the OSGi console with the server use the property -DosgiConsole when starting the server. The INSTALL.txt file found on the installation directory will give you a comprehensive set of options and properties that can be passed into the startup script
  7. Sample configurations can be started by the wso2esb-samples script passing the sample number with the -sn option (Please have a look at the samples guide for more information, on running samples)

What's New In This Release

Login Analytics

Latest IS analytics version comes with the this new feature which enables users to view/analyze all the authentication data where users logged in using SSO scenarios through different protocols such as SAML2 Web SSO, OpenID Connect, Passive STS. This feature will also provide the ability to view authentication data based on different attributes of the authentication events. Furthermore, Users can drill down and analyze data based on following attributes of the authentication flow

  • Service Providers
  • First Login
  • Identity Providers
  • User Roles
  • Users
  • Region

Session Data Analytics

This feature will enable users to view/analyze the evolution of user sessions through Identity Server over time.
